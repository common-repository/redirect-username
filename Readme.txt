== Description ==

Plugin "Redirect username" does not allow to know the login site administrator via the url address type http://example.ru/?author=1.

P.S: If you have any suggestions on how to improve the plugin, please send email to <cite>info[@]artemsannikov.ru</cite>

== Installation ==
	
1. Upload plugin "Redirect username" to the `/wp-content/plugins/` directory;
2. Activate the plugin "Redirect username" through the 'Plugins' menu in WordPress.
